# Code of Conduct

## Our Standards

- Use welcoming and inclusive language
- Be respectful of differing viewpoints
- Accept constructive criticism gracefully
- Focus on what is best for the project

## Scope

This code of conduct applies to all project spaces: issues, pull requests, and discussions.

## Enforcement

Instances of unacceptable behaviour may be reported to conduct@agnos.dev. All reports will be reviewed and investigated.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
